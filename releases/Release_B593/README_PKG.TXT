
This .zip file contains the hardware, software, and documentation
files for this hardware and software version of the NMRA sender board and the
SEND.EXE program.

Once you have extracted the files, look in the following subdirectories
for more information:

BIN       This directory contains the executables and initialization files.
DOC       This directory contains the docs for the hardware and software.
HW        This directory contains the schematic data for the board.
OBSOLETE  This directory contains the retired hardware designs.
SRC       This directory contains the software source code.
