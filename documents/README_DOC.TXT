
This directory contains the documentation files for the NMRA
DCC sender tester.

This directory contains the following files in PDF format:

NMRA TN9.1.1 NMRA DCC Decoder Test User Manual.pdf
	User Manual for the SEND.EXE program.

NMRA TN9.1.2 Sender Board Theory of Operation.pdf
	Theory of operation for the Sender board hardware.

NMRA TN9.1.3 Sender V3 Getting Started Guide.pdf
	Getting started guide for the Sender V3 system.
