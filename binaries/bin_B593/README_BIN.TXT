
This directory contains the software executable and the 
initialization file for the NMRA DCC SEND.EXE program. 
Please refer to the DOC subdirectory for the documentation
on using this program.

This directory contains the following files:

SEND.EXE   DCC Sender board executable
SEND.INI   Sample initialization file for SEND.EXE
S_USER.TXT Output of "SEND.EXE -u" command.
